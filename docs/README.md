## 快速开始

### 1. 业务流程概览

```
用户登录 → 选择工具 → 上传图片 → 配置参数 → 积分校验 → 提交任务
                                                    ↓
任务处理 ← 图片上传 ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
                    ↓
              结果展示 → 下载保存
```

### 2. 核心模块

#### 认证系统
- **自有账号**: 用户名密码登录、手机验证码登录
- **第三方SSO**: 跨平台单点登录（加密Ticket机制）

#### 积分系统
- 积分账户管理
- 任务消耗计算（支持VIP折扣、批量折扣）
- 积分变动动画

#### 任务系统
- 任务提交与状态管理
- 智能轮询（页面可见性+用户活动检测）
- 任务队列与并发控制

